## 反转字符串

**问题**

这个题目说的是，给你一个字符串，你要写一个函数左右反转它。然后返回反转后的字符串。

```markdown
比如说给你的字符串是：

abcde

你要返回左右反转后的字符串：

edcba
```

**代码**

```java
public class AlgoCasts {

  // Time: O(n), Space: O(n)
  public String reverseString(String s) {
    if (s == null) return null;
    char[] c = s.toCharArray();
    for (int i = 0, j = c.length-1; i < j; ++i, --j) {
      char tmp = c[i];
      c[i] = c[j];
      c[j] = tmp;
    }
    return new String(c);
  }

}
```
